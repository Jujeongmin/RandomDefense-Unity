## 필요한 그래픽 자료 (직접 준비)

| 자료 | 규격 | 비고 |
|---|---|---|
| 앱 아이콘 | 512×512 PNG (32bit, 알파 없음) | 필수 |
| 피처 그래픽 | 1024×500 PNG/JPG | 필수 |
| 폰 스크린샷 | 최소 2장, 세로 (권장 1080×1920+) | 필수 — 게임 화면 캡처 |
| 태블릿 스크린샷 | 7"/10" 각 1장 이상 | 태블릿 지원 표시 원할 때 |

스크린샷 추천 장면:
1. 유닛 가득한 전투 장면 (등급 다양하게)
2. 태초/신화 소환 순간 (소환 문구 노출)
3. 보스전 (보스 타이머 노출)
4. 연구소 화면
5. 확률 정보 패널 (투명성 어필)
